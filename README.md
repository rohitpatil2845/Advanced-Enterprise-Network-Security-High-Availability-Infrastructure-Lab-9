# Advanced-Enterprise-Network-Security-High-Availability-Infrastructure-Lab-9

Project Overview
Designed and implemented a secure enterprise network infrastructure for a multi-department organization using Cisco Packet Tracer. The project focused on network scalability, redundancy, security, high availability, wireless integration, VoIP services, and centralized resource management.
The network was built using a hierarchical architecture consisting of Core, Distribution, and Access layers. Multiple departments were segmented using VLANs, while routing and security mechanisms ensured efficient and secure communication across the organization.

Key Features
Designed a hierarchical enterprise network with redundancy at multiple layers.
Configured VLANs for Management, LAN, WLAN, Voice, Server, and Security zones.
Implemented Inter-VLAN Routing using multilayer switches.
Configured OSPF for dynamic route advertisement and fast convergence.
Deployed HSRP to provide gateway redundancy and failover capability.
Implemented EtherChannel using LACP for increased bandwidth and link redundancy.
Built a dedicated DMZ environment hosting Web, Email, FTP, Application, and Storage servers.
Configured Cisco ASA Firewalls to secure internal, external, and DMZ traffic.
Enabled secure SSH-based administration for network devices.
Integrated Wireless LAN Controller and Lightweight Access Points for centralized wireless management.
Configured VoIP services with dedicated voice VLANs and IP phones.
Implemented DHCP, DNS, and RADIUS services for centralized network management.
Established dual ISP connectivity to improve internet availability and business continuity.
Applied ACL-based traffic filtering and security policies.
